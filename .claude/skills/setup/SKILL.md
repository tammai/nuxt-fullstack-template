# Project Setup — Nuxt v4 Fullstack MVP

## Stack

Nuxt v4 · Nuxt UI · Pinia + Pinia Colada · VueUse · Zod · Drizzle ORM · Cloudflare D1 + KV + R2 · nuxt-auth-utils · Cloudflare Pages

## Official Nuxt v4 Docs (LLM-optimised)

Fetch for any unfamiliar Nuxt API, composable, or config option:

```
WebFetch("https://nuxt.com/llms-full.txt", "Find documentation for <topic>")
```

## First-Run Checklist / Kiểm tra lần đầu

```bash
# 1. Install dependencies
pnpm install

# 2. Copy env template and fill in values
cp .env.example .env

# 3. Verify environment — must all pass before writing code
./init.sh

# 4. Generate Cloudflare types (after wrangler.toml changes)
pnpm cf:types
```

## Local Dev Commands / Lệnh phát triển

```bash
# Standard Nuxt dev server (no Cloudflare binding simulation)
pnpm dev

# Dev with Cloudflare D1/KV/R2 simulation via nitro-cloudflare-dev
# Use this when working on server/api/ routes that access bindings
pnpm dev  # nitro-cloudflare-dev is loaded as a Nitro module automatically
```

## Environment Variables / Biến môi trường

Required in `.env`:

```bash
# nuxt-auth-utils — session encryption (min 32 chars)
NUXT_SESSION_PASSWORD=

# OAuth — Google
NUXT_OAUTH_GOOGLE_CLIENT_ID=
NUXT_OAUTH_GOOGLE_CLIENT_SECRET=

# OAuth — GitHub
NUXT_OAUTH_GITHUB_CLIENT_ID=
NUXT_OAUTH_GITHUB_CLIENT_SECRET=
```

## D1 Database / Cơ sở dữ liệu

```bash
# Generate migration files from schema changes
pnpm db:generate

# Apply migrations locally (creates .wrangler/state/d1/)
pnpm db:migrate

# Visual schema browser
pnpm db:studio
```

Schema file: `server/database/schema.ts`
Migrations: auto-generated by drizzle-kit
Local data: `.wrangler/state/d1/`

## Cloudflare Bindings in Code

```typescript
// D1 — Drizzle ORM
const db = useDB(event)  // server/utils/db.ts

// KV
const kv = useKV(event)  // server/utils/kv.ts
await kv.get('key')
await kv.put('key', 'value', { expirationTtl: 3600 })

// R2
const { STORAGE } = event.context.cloudflare.env
await STORAGE.put('avatars/uuid.jpg', body)
const obj = await STORAGE.get('avatars/uuid.jpg')
```

## Verification / Kiểm tra

```bash
./init.sh           # full: install + typecheck + lint + unit tests + build
pnpm typecheck      # type safety only
pnpm lint           # code style only
pnpm test:unit      # Vitest unit tests
pnpm build          # Cloudflare Pages build (output: .output/)
pnpm test:e2e       # Playwright E2E (requires running dev server)
```

All checks must pass before marking a feature done (see CLAUDE.md Definition of Done).

## Troubleshooting / Xử lý sự cố

| Issue | Fix |
|-------|-----|
| `Cannot find binding DB` | Check `wrangler.toml` has `[[d1_databases]]` block, run `pnpm cf:types` |
| Session not persisting | Check `NUXT_SESSION_PASSWORD` is set in `.env` |
| D1 local data missing | Run `pnpm db:migrate` to apply migrations locally |
| TypeScript binding errors | Run `pnpm cf:types` to regenerate `server/cloudflare-env.d.ts` |
